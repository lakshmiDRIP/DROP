# DROP Sample Monte Carlo Integral Package

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP Sample Monte Carlo Integral illustrates R<sup>d</sup> to R<sup>1</sup> Numerical Monte-Carlo Integration.


## Class Components

 * [***SamplingAsymptoticsComparison***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/SamplingAsymptoticsComparison.java)
 <i>SamplingAsymptoticsComparison</i> Compares the Asymptotics of Stratified vs. Uniform Sampling Monte-Carlo Integration of R<sup>d</sup> to R<sup>1</sup> Objective Function.

 * [***StratifiedSamplingAsymptotics***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/StratifiedSamplingAsymptotics.java)
 <i>StratifiedSamplingAsymptotics</i> illustrates the Asymptotics of Stratified Sampling Monte-Carlo Integration of R<sup>d</sup> to R<sup>1</sup> Objective Function.

 * [***StratifiedSamplingAsymptoticsMISER***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/StratifiedSamplingAsymptoticsMISER.java)
 <i>StratifiedSamplingAsymptoticsMISER</i> illustrates the Asymptotics of Stratified Sampling Monte-Carlo Integration of R<sup>d</sup> to R<sup>1</sup> Objective Function using the MISER Sample Allocation Scheme.

 * [***StratifiedSamplingIntegration***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/StratifiedSamplingIntegration.java)
 <i>StratifiedSamplingIntegration</i> illustrates Stratified Sampling Monte-Carlo Integration of R<sup>d</sup> to R<sup>1</sup> Objective Function.

 * [***StratifiedSamplingIntegrationMISER***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/StratifiedSamplingIntegrationMISER.java)
 <i>StratifiedSamplingIntegrationMISER</i> illustrates Stratified Sampling Monte-Carlo Integration of R<sup>d</sup> to R<sup>1</sup> Objective Function using the MISER Sample Allocation Scheme.

 * [***UniformSamplingAsymptotics***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/UniformSamplingAsymptotics.java)
 <i>UniformSamplingAsymptotics</i> illustrates Asymptotics for Uniformly Sampled Monte-Carlo Integration of R<sup>d</sup> To R<sup>1</sup> Objective Function.

 * [***UniformSamplingIntegration***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/sample/mcintegral/UniformSamplingIntegration.java)
 <i>UniformSamplingIntegration</i> illustrates Uniformly Sampled Monte-Carlo Integration of R<sup>d</sup> To R<sup>1</sup> Objective Function.
