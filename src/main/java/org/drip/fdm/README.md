# DROP FDM

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP FDM holds the Multi-dimensional Finite Difference Evolution Schemes.


## Component Packages

 * [***Crank Nicolson***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/fdm/cranknicolson)
 DROP FDM Crank-Nicolson implements the Finite Difference Crank-Nicolson Discretizer.

 * [***Definition***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/fdm/definition)
 DROP FDM Definition holds the Finite Difference PDE Evolver Schemes.
